Quick Start

Copy the one-line installer and run it in iSH:

sh -c "$(curl -fsSL https://raw.githubusercontent.com/statikfintechllc/zsh.installer/master/master/ish.zsh.installer)"
View README

Customize

Change theme or add plugins in ~/.zshrc.

nano ~/.zshrc

Git Credentials

We configure credential.helper store by default (plain-text file).

git config --global credential.helper store
